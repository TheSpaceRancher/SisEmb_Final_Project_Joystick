This is the Final Projecto for the Embebbed System Subject in UPAEP on athum of 2025.

The main objective was develop a full funcional USB Gamepad, using the PIC 18F4550 and USB Libraries, so the proces was the following.
1- Electronic design on Proteus.
2- Main code and library develpoment on Visual studio code.
3- Simulation and functionality test on Proteus.
4- PCB Circuit design on KiCad.
5- PCB manufacturing on CNC.
6- CAD design.
7- 3D Print.
8- Final assembly.
9- UCR remaping to Xinput.
